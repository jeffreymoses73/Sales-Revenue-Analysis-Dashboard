# Sales & Revenue Analysis Dashboard

## Project Overview
This dashboard analyzes sales and revenue data using Power BI.

## Tools Used
- Power BI Desktop
- CSV Dataset

## Features
- Total Revenue KPI
- Total Quantity Sold KPI
- Revenue Trend Analysis
- Top Performing Products
- Revenue by Region
- Interactive Filters and Slicers

## Dataset Columns
- Date
- Product
- Category
- Region
- Quantity
- Unit Price
- Revenue

## Outcome
The dashboard helps identify sales trends, top-performing products, and regional performance through interactive visualizations.
